# 🌦️ Weather App (Python)

This is a simple command-line weather application written in Python. It allows users to check weather details for a few predefined cities.

## 🚀 Features
- Get temperature, humidity, weather condition, and wind speed
- User-friendly input system
- Handles invalid city names gracefully
- Easy to extend with more cities

## 🛠️ Technologies Used
- Python

## 📂 How It Works
- Weather data is stored in a Python dictionary
- User enters a city name
- The program fetches and displays the weather details
